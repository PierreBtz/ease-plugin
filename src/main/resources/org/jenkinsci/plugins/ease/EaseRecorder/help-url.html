EASE publishing web service has URL for each production environment.

If you are not sure which production environment to use, check with Apperian Customer Support at helpdesk@apperian.com.

Environment URL
North America https://easesvc.apperian.com/ease.interface.php
Europe https://easesvc.apperian.eu/ease.interface.php

Here variable expansion $VAR is allowed. Standard jenkins namespace is used thus it could be environment variables or build parameters.
