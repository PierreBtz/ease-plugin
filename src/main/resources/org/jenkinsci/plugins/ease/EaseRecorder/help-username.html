Email address for a valid EASE user in your organization's EASE account.

Email address is an EASE user's unique identifier and is used to log in to the EASE database.

Based on this email address, the EASE web service identifies your organization in the EASE database.

Username and password could be skipped here if there is some credential stored in Jenkins credential plugin. Matching is done in global store and domain that includes url's hostname.

Here variable expansion $VAR is allowed. Standard jenkins namespace is used thus it could be environment variables or build parameters.
